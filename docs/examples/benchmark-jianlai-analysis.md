# 《剑来》开篇对标分析（人类范文基准）

> 用户提供的对标文本：《剑来》第一卷第一章「惊蛰」、第二章「开门」（烽火戏诸侯）。**全文不入库**（版权）；本文只存分析、短句引用与声音指纹。分析用摘录存于 `experiments/jianlai-ch01-excerpt.md` 与 `experiments/jianlai-ch02-excerpt.md`（gitignored，仅本地）。
> 用途：为工作流提供一个"用户认可的人类写法"基准，校准规则与审稿维度。

## 二、被打穿的三个工作流假设

1. **"人味 = 中短句"是错的。** 剑来是 43.8 字的绵长陈述 + 极短收束（"星空璀璨"式短段），方差 0.899 冠绝三方。硬指标从来不是句长，是**方差**；且方差的方向（绵长型 vs 中短型）是书的声音，不是优劣。
2. **"分句连排 = AI 打点"要修正。** 剑来分句复杂度 5.22（"一手持蜡烛，一手持桃枝，照耀房梁、墙壁、木床等处"），比 DeepSeek 高得多。病不在"多"，在"匀"：长短错落、有主有次的连排是中文活水，2-4 字均匀碎片才是打点。
3. **"对白少 = 对白真空"要看章型。** 剑来 ch01 是立章（立人物立世界），对白占比 0.093 与 DeepSeek 持平；拾骨人 ch01 是交锋章（0.211）。指标必须按章型校准，不能一刀切。

## 三、剑来做对了什么（逐条带证据）

1. **开场是"氛围契约"而不是"目标前置"。** "二月二，龙抬头。"六字民俗锚点独立成段，然后讲者现身交代小镇与少年——没有任何"主角此刻想要什么"的明示，处境自己产生压力（官窑关闭、师父新丧、被拒门外）。教训：立章的开场允许讲者型，"句 2 跟上目标与压力"只适用于交锋章。
2. **背景陈述有声音。** "便是陈平安想要当败家子，也无从下手"——交代家徒四壁时顺手给了人物性格与幽默。背景不是罪，没有声音的背景才是罪。
3. **排比与复沓全在人物手里。** 民俗童谣"烛照梁，桃打墙"、睡前念词"碎碎平安，岁岁平安"——形式是排比，归属是民俗与人物，所以是人味高光而非机械。
4. **细节即关系。** "黄土院墙都很低矮……可每次跟陈平安说话，偏偏喜欢蹲在墙头上"——一个动作把两个少年的贫富与心理高低全写完。
5. **机锋长在处境上。** "难道打铁这门活计，不是看臂力大小，而是看面相好坏？"——反讽疑问句，是人物的委屈与聪慧（lint 已据此豁免疑问句与对白中的否定对比）。
6. **场景靠物件链接。** 桃枝→星空→小竹椅→青石板→墙头→绣袋→鱼篓→四脚蛇→木板床：一章九个场景件件有锚，没有一个靠情绪硬转。
7. **伏笔只给画面不给解释。** 四脚蛇"额头上有隆起，如头顶生角"——诡异感点到为止，零解说。

## 四、工作流据此的改动（已全部落地）

- `lint.not-is-flip`：豁免疑问句与对白（陈述句收尾的否定翻转仍 blocking）。
- 场景包模板：**章型字段**（交锋/立章/过场/收束），立章放宽信息预算、不可逆选择可写"无（立章）"，审稿与指纹按章型校准。
- texture-editor：维度 1/2/4/7 全部"功能化"（堆叠看匀不看多、排比先问是谁的、陈述分有声无声、按章型校准）；新增"机锋是合法资产"。
- 反模式文档：新增「功能合法化」附录（五种"两种"，含剑来原文例证）。
- voice-bible 正面引导：机锋合法。

## 五、悬而未决（下阶段观察）

- **章型的指纹基准库**：立章（剑来 ch01）、交锋（拾骨人 ch01）、过场+交锋混合（剑来 ch02）已有基准；收束章型仍缺人类样本。
- **绵长型声音的生成**：剑来式 40+ 字长句的写作纪律（长而不糊）目前只进了分析，没进模板；如果新书定调偏绵长，voice-bible 的节奏蓝图需要另一种默认。
- **lint 在剑来身上的 advisory 噪音**（已处理）：`rhythm-monotony` 曾把剑来的"一句一段"误判为单调——已加句长 CV 门槛（run 内句长 CV≥0.45 豁免，剑来两段实测豁免、DeepSeek 式均匀短段仍命中）。`word-count-tic` 命中的"某四个字""说了两个字"属功能计数（数量对画面负责），非规则误判；规则消息已从"建议改为非数字表达"改为提问式（"若计数不对情节或画面负责，建议删去或改写"），功能判断交审稿层。

## 六、第二章实测：四个猜想的验证（2026-07-17）

用户猜想：(1) 剑来与工作流产出只是风格不同；(2) 被 AI 鉴定拦下的东西，人类可能认为是好的；(3) 剑来这种书第二章随时会变调；(4) 人类把技巧用到缺点看不出，这正是人机的区别。全部用 ch02 实测验证。

### 6.2 lint 对剑来两章的实测

- ch01：**1 blocking**（L87 `ellipsis`："碎碎平安，岁岁平安……"——全书最著名的收尾句）+ 2 advisory（rhythm-monotony、colon-density）。
- ch02：**0 blocking** + 6 advisory（rhythm-monotony ×3、word-count-tic ×2、colon-density）。
- 另外两条 L1 硬禁令（`em-dash`、`not-is-flip`）对两章**零命中**——烽火这两章根本不用这两种形式；ch01 的"不是看臂力大小，而是看面相好坏？"是疑问句，豁免规则正常工作。

### 6.3 裁决

1. **风格不同 → 成立，但不能全用"风格"解释。** 剑来两章自相像（句长 44、分句 5.1、比喻 1‰ 上下），与拾骨人（17.5/2.8/0.6‰）确是两种声音（绵长 vs 冷硬）。但跨风格存在人类共性：比喻克制（≤1.5‰ vs DS 3.51‰）、有人物用问句推进（0.06–0.14 vs DS 0.008）、分句有主有次（2.8/5.2 vs DS 1.4 匀质）。DS 的问题不是风格不同，是**没有风格**——所有指标全扁。
2. **AI 鉴定拦下的可能是人类的好东西 → 成立，铁证在手。** 剑来 ch01 最出名的一句带省略号，被 blocking 拦下：烽火走我们的流水线，第一章交不了稿。规则判形式，读者判功能——那个省略号是"念叨声渐弱淡出"的功能，不是偷懒。但硬禁令整体是准的（另两条零误伤）；省略号维持禁用是脚手架选择：**AI 用 …… 九成是装淡出，等真写出"非它不可"的句子再破例**。决策权留用户。
3. **第二章会变调 → 成立，但变的是配比不是声音。** 声音主干几乎不动（句长 43.8→44.8、分句 5.22→5.09、比喻 1.34→1.05），动的是场景功能决定的配比：问句率 0.059→0.141（城门斗嘴全靠问句）、碎段率 0.136→0.060。**章型决定配比，口吻贯穿全书**——场景包的章型字段管的正是这个；剑来 ch02 同时补上了"过场章型"的人类基准（此前空缺）。
4. **人类熟练到缺点看不出 → 基本成立，修正为"缺点被声音与动能收编"。** ch02 里套话（"豪气干云""心不跳气不喘"）、解释腔（"理由很简单，……"）、排比（"有人失望，有人玩味，有人皱眉，有人讥讽"）全在场，全都活着：每个都有 pov 归属或当场回报（套话接在拍铜钱的动作后，读作人物口气而非作者凑词）。AI 裸用同样的形式，所以刺眼。连转载文本里"刘羡阳/刘阳羡"名字打架（疑似网站 OCR 错字），读者也照滑过去——动能在，瑕疵被带着走；AI 没有动能也没有声音，所以每个瑕疵都是裸露的。
