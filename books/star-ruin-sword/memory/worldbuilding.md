# 世界设定

## 物理规则
- 宇宙中存在"道脉"（Dao Meridian）——连接恒星系的亚空间通道，由密集的天然灵脉构成。道脉在物理上不可见，只有修行者能通过灵力感知其"流"（current）。
- 道脉不恒定：其流向、强度、稳定性随时间波动。大型道脉稳定期以千年计，支流道脉可能在数十年内偏移或消散。
- "灵压"（spiritual pressure）是道脉的核心物理量。工业仪器可以测量宏观灵压变化（精度约0.1灵压单位），但无法检测修士级别的精细灵力波动（精度需达到0.001以下）。
- 超光速航行依赖道脉。无道脉连接的两个星系之间，没有任何技术可以逾越——这就是道脉控制即政治权力。
- 规则限制：不存在"传送阵"或"空间折叠"技术。所有航行必须经由道脉，航行时间取决于道脉长度与流速。铁棘号从渊流航道到最近港口"黑石站"需要约三周亚空间航行。

## 社会规则
- 星盟公约（Star Alliance Accord）是银河系的核心政治框架。修行宗门在星盟理事会中占有历史席位，但实际权力已被企业联盟侵蚀。大宗门仍是道脉的"法定维护者"，但日常航运调度由企业说了算。
- "道脉领航员"（Meridian Pilot）是灰色职业：他们在宗门体系外受训，灵力水平不足以正式入宗修行，但恰好够感知大型道脉的流向。被航运公司雇佣，按趟计酬，不享受船员待遇。法律地位模糊——算"设备操作员"而非"修士或船员"。
- 剑修身份受星盟《武器管制公约》限制：任何具备"定向灵力攻击能力"的修士必须在星盟登记。未登记的剑修=非法武装。青霄剑宗覆灭后，所有青霄弟子理论上应向星盟执法部申报转移登记——没有人报过。法律上青霄弟子仍是"未登记武装人员"。
- 泰衡星际（Taiheng Interstellar）：银河系最大的航运与矿业企业，控制约17%的已知道脉网络。其安全部有自己的武装舰队和情报网络，在偏远星系的权力远超星盟执法部。
- 日常语言：通用语（星盟官方语言）是汉语系，各星系有方言。修士之间有古语称谓（道号、师承名），普通人不用。
- 货币：灵晶（spirit crystal）硬通货，同时用作工业能源和修行资源。普通工人月薪约2-3单位灵晶。一趟道脉领航收入约0.5单位灵晶。

## 道脉技术
- "灵压检测仪"（Pressure Gauge）：每个道脉航道的关口站安装的工业仪器。测宏观灵压，精度有限。
- "导航矩阵"（Nav Matrix）：货船的核心控制系统，接收领航员的灵力输入并转化为航线指令。有安全锁——只有船长授权才能变更航线。
- "灵压记录仪"（Pressure Recorder）：每艘道脉航船的黑匣子组件，持续记录导航舱的灵力输入波形。可用于事后分析，也可用于法律取证。

## 禁忌
- 禁止在公开场合谈论"脉崩协议"——这个词只在泰衡安全部内部使用。外部任何人说出口等于承认自己接触了机密。
- 修士不能在道脉航道中释放超过0.5灵压单位的灵力——高灵力输出会扰动航道，可能导致局部崩解。陆渊的空明九斩瞬间输出超过3.0灵压单位。这是重罪。
- 道脉领航员不能触碰导航矩阵以外的任何船载系统——雇佣合同明文条款，违反即解雇+起诉。
