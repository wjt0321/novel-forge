## narrative_distance
第三人称有限视角，锚定陆渊。读者只感知陆渊能感知的——不跳入任何其他角色的内心，不以叙述者口吻解释世界。

动作场景：收紧到"当下身体感知"级别（体温、肌肉、灵力流向）。
静态场景：维持在"选择性注意"级别（陆渊注意到了什么，忽略了什么，为什么）。

## focalization
全程陆渊。他对世界的认知受限于：他能感知的灵力波动、他作为前剑修的专业训练（对异常的警觉）、他作为底层领航员的社会经验（知道何时沉默）。

他还不知道的：泰衡的"脉崩协议"、道脉的真实起源、星髓矿的真正用途。

## 节奏蓝图（每章写作前必读）

一章的标准节奏不是均匀的——它应该像呼吸一样有起伏：

```
[开场] 密集建立空间与身体 → 3-5段，用具体物象锚定读者的感官
[升温] 对话交锋，节奏加快 → 2-3段，冲突通过对话—动作—对话推进
[慢拍] 纯粹感知段落 → 1-2段，不推进情节、不给新信息、不建立冲突
[加速] 危机逼近 → 短段落，灵力触觉先行，视觉随后
[高潮] 不可逆动作 → 每个句子只承载一个动作或一个感知
[收束] 静态收束 → 回到身体——呼吸、心跳、冷、疼
```

**人物性呼吸段**：不以字数或数量配额强制插入停顿。需要减速时，呼吸段必须服务于人物：回避、拖延、误读、身体失控、关系余温或价值暴露；具体功能须记录在 `planning/scene-package-chXX.md`。它可以不推进主线，但不得偷渡新的关键情节或退化为无关填充。

## sentence_rhythm

动作场景：短句。灵力触觉先行，视觉随后。例如——"灵压骤降。像脚底下的地板突然缺了一块。"

感知场景：中长句，但每句只承载一个鲜明物象。当你发现自己在一句话里塞了两个以上的意象时，拆成两句。

对话场景：节奏由说话人此刻想说什么、不想说什么、被什么打断了来驱动——不是由"他是命令型还是保留型"的标签驱动。

当你想用三个以上结构相似的短句（"他感觉到……他看见了……他意识到……"）时——删掉其中两句。用物象替代情绪。

## 角色语言指纹（替代旧版「对话规则」）

不要用标签写对话（"船长=命令句"）。每个角色有自己的语言指纹：

**陆渊** — 他的句子偏短，习惯性省略主语。但这不是规则，是症状。他省略主语不是因为"他是保留型"，而是因为他五年没跟人说过心里话——每次开口前他都在心里先删掉一半。他的紧张不在句子里，在句子之间。沉默越长，接下来的话越短。

**方镇** — 他说话像在填航海日志——每个词都是可被审计的。他不会说"我觉得"，但会说"数据显示"。当他说"那就"被打断时，那个未完成的"那就"可能是他那天唯一一次差一点说出非标准用语。他的不信任不是个人恩怨，是制度化的——他是制度的人格化身。

**柯晴** — 句末常用"啊""吧"，语气看似随便但信息密度高。她是唯一会在陆渊面前吃完话又补一句的人——不是因为她友善，是因为她需要准确，而准确有时需要第二句。她把弹射表塞进陆渊口袋时纸是温的——这个动作本身就是她的语言。

**姜凝**（第2章出场）— 句末句号多，几乎不用语气词。问句像陈述——审问性。她的问题不需要答案，答案她已经知道了，她需要的是你的反应。

### 对话的铁律（仅2条，其余正面引导）

- **台词卡禁止**：连续四句以上仅有引号包裹、无任何动作/心理/场景穿插的纯对白 → MUST级错误。
- **归属感强制**：三句以上无说话人标识的短对话连续出现 → MUST级错误。这不是格式问题，是读者不知道谁在说话的问题。

## sensory_palette

灵力触觉是核心感官通道——区分温度、压力、节奏、纹理，而不只是"感觉到灵力"。
- 正确示范："道脉的流像冰下暗河"（温度+压力）、"灵力在丹田里像碎玻璃搅动"（纹理+疼痛）
- 错误示范："他感觉到一股强大的灵力波动"

视觉：偏冷色调，偏暗。工业船舱的荧光、道脉裂隙的暗紫色、导航矩阵的冷蓝色。
听觉：机械声要具体到频率——"循环泵每七秒敲一次"，而不是"机器的轰鸣"。
嗅觉：船舱的气味——回收空气的金属味、润滑油、人体气味。不诗化。

## emotional_restraint

陆渊的情绪被五年生存压力压缩成了身体反应。用生理变化+决定+行动延迟来呈现情绪，而非内心独白式的总结。

当你想要写"他感到愤怒/绝望/悲哀"时——停下来。问他：此刻他的身体哪里在变化？他的手在做什么？他选择了不说哪句话？

如果陆渊的愤怒表现出来了，那只能是因为他选择了让它表现出来——或者他控制不住了。

## 硬禁令（仅保留3条）

1. 禁止"——"和"……"（em-dash 和中文省略号）。
2. 禁止"他意识到/他明白/他忽然理解了"之类的解释性导语——用身体反应来呈现认知。
3. 禁止连续段落跨段近义重复——相邻两段以不同措辞重复同一信息。

## 正面引导（替代原禁令4-7条）

- **替代"不是X而是Y"**：当你想用否定来强调时，试试用一个具体物象来替代否定句。不说"不是仪表能测出的不对"，而说"仪表盘平静如水，他的指腹贴在玻璃上感受着水下的暗涌"。
- **替代"仿佛在说/似乎在宣告"**：如果场景在"宣告"什么，换成角色注意到了什么。不说"那声音仿佛在宣告他的死亡"，而说"他听到那个声音。然后他注意到自己的手在解锁安全扣。"
- **替代感叹号**：非对白段落中的感叹号99%是在替代你找不到的准确词汇。删掉感叹号，重新找那个词。
- **替代科技套话**："冷光闪烁""数据流如瀑布"——删掉。用你在这间舱室里真正能看到、听到、摸到的东西替换。

## 写前仪式：角色独白

在正式动笔写正文之前，以陆渊第一人称写300-500字的独白。这段独白不会进入正文，目的是让叙述者从"第三人称观察者"切换到"第一人称体验者"。
独白不需要质量，只需要存在。问自己：此刻陆渊最不想想的是什么？让他去想那个。

## exemplar_notes

第一章开场定位：不从"宇宙的宏大"写起。从陆渊的导航舱写起——一个三平方米的金属空间。他的世界首先是这间舱室，然后才是舱室外的道脉。用他的身体状态引入世界观：后背贴在人造革座椅上，汗水让椅面发黏。他感觉不到道脉的流了——不是消失，是变了。
