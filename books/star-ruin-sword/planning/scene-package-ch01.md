# Scene Package — 第一章「渊流崩解」

## 0. 边界

- 承接上文：开篇场景，无上文。
- 本场景从何处开始、在何处停止：陆渊在导航舱察觉异常，到被列入第四十七号救生舱位、准备弹射。
- 允许新增的长线谜团：三箱星髓矿被裂隙完整吞没。
- 不得在本场景解决的问题：五年前道脉崩塌真相、陆渊的指控与救生舱人数缺口。

## 1. 场景压力

- **视角角色此刻想要什么：** 安全完成领航，不惹麻烦地回到黑石站。
- **若什么都不做会失去什么：** 铁棘号和船上人员会被崩塌面吞没。
- **最直接的阻力：** 仪表正常、船长方镇拒绝偏航、崩塌正在加速。
- **阻力背后的人的诉求：** 方镇必须遵守调度与责任链，避免无授权改线。
- **不可逆选择：** 陆渊用空明九斩强行接入灵核，将船撕入未标注支流。
- **选择立即造成的后果：** 船获救，陆渊暴露剑修手段并被电击锁环控制。
- **章末遗留的具体压力：** 他掌握异常共振和矿箱消失的线索，却面临导航劫持指控及不可信的救生名单。

## 1b. 情感弧（可选）

- **开场情感状态：** 职业性警觉压住疲惫；他先盯住仪表和周期，不愿触碰五年前的记忆。
- **不可逆选择时刻的情感状态：** 没有英雄冲动，只有在倒计时内拒绝重演一次沉默的决断。
- **章末残余情感状态：** 身体仍在疼，注意力却落在第四十七号、马达差频和无法删除的记录上。
- **本场不应替角色解释的情绪：** 不写陆渊“感到解脱、悲壮或愤怒”；由他不辩解、数差频和攥住纸张呈现。
